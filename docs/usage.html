Download, unzip, and run:

  • Download the CAL-JLC application from this site.
  • Unzip the downloaded CAL-JLC_08.zip file to a location of your choice, keeping all of the file contents together.
  • Double-click the CAL-JLC_08.exe file to launch the application.

On first use, you will be prompted to enter some connection and configuration information:

  • If the JLC panel is using its default settings, its address is 192.168.200.114, using port 49300.
  • Check your mixer's UI to setup/view its CSCP address and port, ensuring CSCP is enabled.
  • Enter "y" at the prompt for two-way fader control, o "n" for one-way JLC->CAL
  • Choose any faders from 1 to 192 on the mixer to map to the 8 faders on the JLC panel. Note, mixer CSCP faders "wrap" so you can control different layers / do not have to control the visible/top layer, ... e.g. on a mixer with 32 physical faders, CSCP fader 33 = Layer 1 B fader 1, CSCP fader 65 = Layer 2 A fader 1 etc.).
  • Ensure the machine this appication is running on has IP address/es that are reachable by the addresses set on both the mixer and the JLC.

JL-Cooper panel configuration can be changed by accessing its setup UI using a web-browser on its address. To set the JL-Cooper panel back to its default address (192.168.200.114) and configuration, press and hold the 3 white buttons in fader strip 8 while powering up the unit... The unit will indicate when it has reset and will prompt to repower for the changes to be applied.

Current - CAL-JLC version V0.8 (for beta evaulation). Tested with Calrec Brio and JL-Cooper Eclipse MXL2.

This version provides 2-way control of faders and cut/on buttons, along with fader labels passed at start-up.
