CAL-JLC can be evaluated for free. With no license installed, the application will run for a limited amount of time. For pricing to obtain a license for unlimited use on a given machine, email contact.caljlc@gmail.com your product key (which is displayed within the application when running in evaluation mode, and copied to your clipboard so it can be easilly be pasted into an email).

Download, unzip, and run:

  • Download CAL-JLC
  • Unzip the downloaded CAL-JLC_1.zip file and keep all of the file contents together in the same folder.
  • Double-click the CAL-JLC_1 exe/application file to launch.


On first use, you will be prompted to enter some setup info

  • Check your mixer's UI to setup/view its CSCP address and port, ensuring CSCP is enabled.
  • If the JLC panel is using its default settings, its address is 192.168.200.114, using port 49300.
  • Ensure the PC this appication is running on has IP address/es that are reachable by the addresses set on both the mixer and the JLC panel.
  • Enter "y" at the prompt for two-way fader control, or "n" for one-way JLC to CAL
  • Choose any faders from 1 to 192 on the mixer to map to the 8 faders on the JLC panel. Note, mixer CSCP faders "wrap" so you can control different layers / do not have to control the visible/top layer, ... e.g. on a mixer with 32 physical faders, CSCP fader 33 = Layer 1B - fader 1, CSCP fader 65 = Layer 2A - fader 1 etc.).


JL-Cooper panel config

View/chang the JLC panel's settings by entering its address into a web-browser. To reset the JL-Cooper panel back to its default address 192.168.200.114 and other settings, press and hold the 3 white buttons in fader strip 8 while powering up the unit... The unit will indicate when it has reset and will show when to repower for the changes to be applied.
