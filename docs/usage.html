Download, unzip, and run:

  • Download CAL-JLC
  • Unzip the downloaded CAL-JLC_08.zip file and keep all of the file contents together in the same folder.
  • Double-click the CAL-JLC_08 .exe/application file to launch.

On first use, you will be prompted to enter some connection and configuration information:

  • If the JLC panel is using its default settings, its address is 192.168.200.114, using port 49300.
  • Check your mixer's UI to setup/view its CSCP address and port, ensuring CSCP is enabled.
  • Enter "y" at the prompt for two-way fader control, o "n" for one-way JLC->CAL
  • Choose any faders from 1 to 192 on the mixer to map to the 8 faders on the JLC panel. Note, mixer CSCP faders "wrap" so you can control different layers / do not have to control the visible/top layer, ... e.g. on a mixer with 32 physical faders, CSCP fader 33 = Layer 1 B fader 1, CSCP fader 65 = Layer 2 A fader 1 etc.).
  • Ensure the PC this appication is running on has IP address/es that are reachable by the addresses set on both the mixer and the JLC panel.

JL-Cooper panel config can be viewed/changed by entering its address into a web-browser. To reset the JL-Cooper panel back to its default address 192.168.200.114, port 49300 and other settings, press and hold the 3 white buttons in fader strip 8 while powering up the unit... The unit will indicate when it has reset and will show when to repower for the changes to be applied.
